Keyboard Shortcuts Save 1 000+ clicks a day.

Launch & Switch
  • ⌘ Space → type “Safari” → Return. Goal: open any app < 2 s.
  • ⌘ Tab → cycle running apps.
  • ⌘ → cycle windows in current app.
  • ⌘ H → hide current app.
  • ⌃⌘ Q → lock screen.
Window & Desktop Control
  • ⌃ ↑ → Mission Control overview.
  • ⌃ ↓ → App Exposé.
  • F11 → Show Desktop.
  • Hover green button → “Tile Left/Right”.
Stage Manager (Ventura+)

Toggle in Control Center, then ⌃ ←/⌃ → to hop stacks.

Presenter Preview (Sequoia 15)

Keynote → “Presenter Preview” mirrors audience view.

Pro Moves
  • Space → Quick Look; press ⌘ Y to full-screen.
  • ⌃⌘ Space → Emoji & Symbols palette.
  • ⌥⇧ Volume ▲/▼ → fine-step volume.
  • ⌘ ⌥ Esc → Force-Quit window.
  • ⌘ 1/2/3 inside iPhone Mirroring (Sequoia).

Trackpad Gestures Navigate three times faster.

Tap

Two-finger tap → right-click menu.

Swipe

Three-finger swipe ↑ → Mission Control.

Edge Pull

Swipe from right edge → Notification Center.

Spread & Pinch

  • Spread thumb + three fingers → Show Desktop.
  • Pinch thumb + three fingers → Launchpad.
  • Spread four fingers → Show Desktop (if enabled).
  • Pinch four fingers → Launchpad (legacy option).

Mirror

Control Center → “iPhone Mirroring”.

Advanced Gestures

  • Rotate with two fingers → rotate images/PDFs.
  • Double-tap with two fingers → smart-zoom.
  • Assign corner swipes in Settings for Sleep or Launchpad.

Finder Workflow Minutes now, hours saved later.

Sidebar Hygiene

Finder → Settings ⌘ , → untick unused drives.

Tags & Smart Folders

Tag file “Action” → File → New Smart Folder → Tag is Action.

Quick Actions

Select image → Rotate Left or combine PDFs instantly.

Batch Rename

Select group → right-click → Rename to add counters or dates.

Quick Look + Mark-up

Press Space, then click the ✏️ to sign PDFs or crop images.

Column View + Preview

Toggle ⌘⌥ P to show metadata while arrow-keying files.

AirDrop Drag

Drag files to another Mac/iPhone in sidebar—no cables.

iCloud Moves Stay synced, keep storage lean.

Manage & Optimize
  • System Settings → iCloud → Manage Storage.
  • Delete retired backups, enable “Optimize Mac Storage”.
File Size Audit
  • ⌘⇧ I → iCloud Drive, ⌘ 2 list view, sort by Size.
Shared Albums
  • Photos → Settings → Shared Library → invite.
Keychain Audit

Settings → Passwords → Security Recommendations.

Hide My Email

In Mail, choose “Hide My Email” in the From field for a burner address.

Custom Email Domains

iCloud.com → Custom Email Domain → follow DNS prompts.

Family Sharing Storage

Upgrade to 200 GB and share space with family for less.

Messages in iCloud

Toggle Messages under iCloud so old attachments sync (then delete to free space).
